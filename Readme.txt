* Download and Install R from [https://cran.r-project.org/] [For windows]
* Set the path to R [for example, C:\Program Files\R\R-*.*.*] in system environment 
* Install GDAL:
	* conda install -c conda-forge gdal
* Install necessary libraries:
	* pip install rpy2==3.4.5 
	* pip install statsmodels
	* pip install matplotlib
